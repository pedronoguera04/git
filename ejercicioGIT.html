  1. ¿Que es un sistema de control de versiones?

    Es una herramienta que registra los cambios realizados en un proyecto a largo del tiempo, permitiendo revertirlos, colaborar con otros desarrolladores y gestionar diferentes versiones del código

  2. ¿Cual es la diferencia entre git y github?

    Git es un sistema de control de versiones que gestiona los cambios en el código de manera local o remota. GitHub es una plataforma basada en la nube que utiliza Git para alojar repositorios, facilitando la colaboración y el acceso al código

  3. ¿Cómo creo una rama en git?

    Con el comando git branch nombre-rama

  4. ¿Cómo clono un repositorio?

    Utilizando el comando git clone URL-del-repositorio

  5. ¿Cómo puedo bajar los ultimos cambios de un repositorio?

    Utilizando el comando git pull

  6. ¿Cómo puedo gener un tag?

    Utilizando el comando git tag y el nombre del tag

  7. Cómo hago una mezcla de ramas?

    Antes que nada nos tenemos que asegurar de estar en la rama donde queremos combinar los cambios, después utilizamos el comando git merge nombre-rama

  11. ¿Cómo puedo ver los cambios que estan por enviarse al repositorio?
